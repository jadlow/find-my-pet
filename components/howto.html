How to use Find My Pet

Report a Lost Pet

  1. Log in if you have an account. If not, please sign up.

Search for Lost/Found Pets' Posts

  1. Navigate to the main page.
  2. Specify the search parameters
    • Specify a coordinate location, to be the center of your search region.
    • Set the radius of the search region.
    • Choose if you are searching for lost pets, found pets, or both.
    • Select the pet type.
  3. Click the search button
